Install/Uninstall

     

Installing Killink CSV:

 

Follow the steps below to install Killink CSV:

 

1.Run the program setup-killinkcsv.exe.
2.Follow the Install Wizard instructions as they appear on the screen.

 

Uninstalling Killink CSV:

 

Go to Add/Remove Programs found in the Control Panel.  From the Install/Uninstall tab, scroll through the list of installed programs.  Select White Peak Killink CSV and click the Add/Remove... button.  Follow the instructions on the screen to complete the uninstall process.

 

Performing a Silent Install:

 

Silent installs are useful when deploying Killink CSV on multiple machines manually or from a deployment script. The Killink CSV setup program includes a silent mode and the Killink CSV program can be quietly registered. Here is what you need to do to silently install and register Killink CSV Editor.

 

Execute the two following statements on the command line or in the deployment script:

 

setup-killink.exe /VERYSILENT

 

"C:\Program Files\WhitePeak\KillinkCSV\killinkcsv.exe" QUIETREGISTER Licensed To nnnnnn-nnnnnnn-nnnnnn-nnnnnnn-nnnnnn-nnnnnn-nnnnnn

 

The first command line statement installs Killink CSV Editor to the default location, which is "C:\Program Files\WhitePeak\KillinkCSV\".

 

The second command line statement registers the software with specified license information. Replace "Licensed To" with the license name and replace "nnnn-nnnn..." with the license key. The license name and key are available in the order confirmation email you received shortly after placing the order.

 

With those 2 command line statements you should be able to deploy Killink CSV Editor and register the site license on every desktop in your organization without having to manual visit each computer.
