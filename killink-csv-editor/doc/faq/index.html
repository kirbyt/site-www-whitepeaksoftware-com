Frequently Asked Questions

      

Setup/Installation

 

Do I need the .NET Framework? No.  Killink CSV is a native Windows program.  You do not need to download any additional frameworks or libraries.

 

Can I uninstall Killink CSV? Yes, absolutely.  While we hope you find Killink CSV useful we understand it is not needed by everyone.  You can uninstall the program by selecting the Uninstall icon in the Killink CSV program group or by using the Add/Remove Programs control panel applet.  Please read the help file for step-by-step instructions.

 

 

General Questions

 

Why the name Killink CSV? Killink is the name of a ski trail at Killington in Vermont.  We tend to select trail names as product names as a symbol of our love for snowboarding.

 

What is Killink CSV? Killink CSV is a spreadsheet editor for comma separated value, tab, and other delimited value text files.

 

Who is the typical Killink CSV user? Anyone needing to view, edit, and print delimited text files.

 

Is there a limit to the number of rows and columns supported by Killink CSV? There is no set limit.  Killink CSV is only limited by the amount of available computer memory.  The more memory available in the computer, the more rows and columns that can be displayed. 

 

Does Killink CSV support Unicode files? No, not at this time.

 

Killink CSV hangs or is slow to start? It's possible the configuration file for Killink CSV is corrupted.Â  Try this to see if it fixes the problem.

 

Open Windows Explorer and navigate to the following directory:

 

C:\Users\%USER-NAME%\AppData\Roaming\White Peak Software\Killink CSV

 

%USER-NAME% is your Windows login name. If this path does not exists then it might be at one of these other locations:

 

C:\Users\%USER-NAME%\AppData\Local\White Peak Software\Killink CSV

 

C:\Users\%USER-NAME%\AppData\LocalLow\White Peak Software\Killink CSV

 

If you are running Windows XP or earlier, the directory is at:

 

C:\Documents and Settings\%USER-NAME%\Application Data\White Peak Software\Killink CSV

 

In this directory, there is a file named KillinkCSV.exe.config. Depending on the configuration of your computer, Windows Explorer may show the file as KillinkCSV.exe instead of KillinkCSV.exe.config. Rename this file, so it can be used if needed. After renaming the file, launch Killink CSV. The problem you are experiencing will go away if a corrupted configuration file is indeed the cause of the problem.

 

Note: The corrupted configuration file means you will lose any customizations you made to Killink CSV such as adding new file types. You must re-apply any customizations you previously made. Sorry for the inconvenience.
